# 👩‍💻 Mon premier Portfolio (Projet archivé)

Bienvenue sur le dépôt de mon **premier portfolio**. Ce projet a été réalisé afin de présenter mon parcours, mes compétences et mes premières réalisations en tant que **Conceptrice Développeuse d'Applications**.

> **⚠️ Ce portfolio n'est plus maintenu ni mis en ligne.** Il est conservé sur GitHub à titre d'archive afin de témoigner de mon évolution et de mon parcours.

---

## 🚀 Projets présentés

* **AS Porcelette** : Site web pour le club d'arts martiaux de Porcelette, comprenant un site vitrine ainsi qu'un back-office avec un système de comptabilité.
* **ADEM** : Site de présentation du rapport annuel d'activité 2023.
* **AdminMnsV1** : Application de gestion administrative développée en ASP.NET MVC avec SQL Server.
